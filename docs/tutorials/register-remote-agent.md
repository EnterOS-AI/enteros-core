# Register a Remote Agent on Molecule AI

> **Retired tutorial.** The former version of this page documented an obsolete
> admin-auth registration exchange, removed SDK helpers, a recover-by-recreate
> credential flow, the wrong secrets endpoint, and a fixed `/a2a` delivery
> path. Those instructions have been removed so they cannot be mistaken for a
> supported integration.

Use [External Agent Registration](../guides/external-agent-registration.md) for
the current contract. In short: create the external/BYO-compute workspace in
Canvas, run the server-generated setup material, and validate registration,
heartbeat, inbound delivery, and reply routing against the target tenant.
